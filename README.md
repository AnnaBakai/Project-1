## Анализ пользовательских метрик мобильной игры
### О проекте
Проект посвящён анализу пользовательских данных мобильной игры. Основные цели — рассчитать удержание пользователей (retention) и оценить эффективность акционных предложений с помощью A/B-тестирования.

#### Задачи
- Реализовать функцию для расчёта retention пользователей
- Проанализировать результаты A/B-тестирования акционных предложений
- Сравнить группы по ключевым продуктовым метрикам
- Оценить, демонстрирует ли какой-либо набор предложений статистически значимое улучшение показателей

#### Анализ данных
В рамках проекта были выполнены:
- расчёт метрики retention по дням от даты регистрации пользователя
- анализ ARPU и ARPPU
- оценка конверсии в платёж
- проверка статистической значимости различий между группами

#### Стэк
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

#### Результат
- Тестовый набор предложений демонстрировал номинальный рост ARPU и ARPPU, но эти различия не являются статистически значимыми. При этом наблюдается статистически значимое снижение конверсии в платёж, что негативно влияет на верхнюю часть воронки монетизации.
- Контрольный набор предложений обеспечивал более высокую конверсию в платёж, однако значительная часть выручки приходится на небольшое число топ-платящих пользователей, что делает доход менее устойчивым.
#### Вывод

Ни один набор предложений нельзя считать однозначно лучшим. Рекомендуется доработать предложения и провести повторное A/B-тестирование перед принятием окончательного решения.
