## Анализ пользовательских метрик мобильной игры
### О проекте
В проекте проведён анализ пользовательских данных мобильного приложения. Основная цель — оценить удержание пользователей (retention) и определить наиболее эффективный набор акционных предложений на основе результатов A/B-тестирования.

#### Задачи
- Реализовать функцию для расчёта retention пользователей
- Проанализировать результаты A/B-тестирования акционных предложений
- Сравнить группы по ключевым метрикам
- Определить наиболее эффективный вариант и обосновать выбор

#### Анализ
В рамках проекта:
- рассчитана метрика retention для оценки удержания пользователей
- проведён анализ результатов A/B-теста
- выполнена проверка статистической значимости различий между группами
- сделаны выводы о наиболее эффективном наборе акционных предложений

#### Используемые инструменты
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

#### Результат
Определён наиболее эффективный вариант акционного предложения на основе анализа продуктовых метрик и статистических тестов.
